# Parallel Programming Assignments

This repository contains the homework assignments for the Parallel Programming course.

## Assignments

### HW1: Parallel Graph Search (Zobrist Hashing)
- **Topic**: Parallel state-space search (likely for a puzzle game) using Pthreads and TBB.
- **Key Technologies**: C++, Pthreads, TBB (Threading Building Blocks), Zobrist Hashing.
- **Slides**: [HW1 Presentation](https://docs.google.com/presentation/d/1VP9ayXHGgwWnLgqoPRv_2m0wrXJnaa_XrdugJOLhPOA/edit?slide=id.p#slide=id.p)

### HW2: Distributed SIFT Feature Extraction
- **Topic**: Implementation of the Scale-Invariant Feature Transform (SIFT) algorithm distributed across multiple nodes.
- **Key Technologies**: C++, MPI (Message Passing Interface).
- **Slides**: [HW2 Presentation](https://docs.google.com/presentation/d/1J341rOmFrNEI5G4utK-kY5C6TFteZJDCxXTSPi7xV00/edit?slide=id.g38e82869866_2_106#slide=id.g38e82869866_2_106)

### HW3: Parallel Mandelbulb Renderer
- **Topic**: Rendering the 3D Mandelbulb fractal using GPU acceleration.
- **Key Technologies**: CUDA, C++.
- **Slides**: [HW3 Presentation](https://docs.google.com/presentation/d/1eYYuvNN8AfPxwFW78Denj5wRgcSk50NGCVrksrwtIkg/edit)

### HW4: Parallel Blockchain Mining
- **Topic**: Parallelized SHA256 hashing to solve a Bitcoin block (Merkle root calculation).
- **Key Technologies**: CUDA, C++.
- **Slides**: [HW4 Presentation](https://docs.google.com/presentation/d/1kcQVTeK10gZ7FUIpvGFD065wgVAzelMcMuWb2HZ5WXU/edit)

### HW5: N-Body Simulation
- **Topic**: Simulation of the N-body problem (gravitational interaction between particles) using heterogeneous compute.
- **Key Technologies**: HIP (Heterogeneous-Compute Interface for Portability), C++.
- **Slides**: [HW5 Presentation](https://docs.google.com/presentation/d/1VEEOqFu_C2LZ3DGcBu8vVA1Un-MwlyZU/edit?slide=id.p1#slide=id.p1)
